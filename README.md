# タップ記録 (Tap Log)

ワンタップで「番号＋時刻(HH:MM)」を記録する、シンプルな作業ログアプリです。
HTML1ファイルだけで動作し、サーバー不要・オフライン対応。Androidのホーム画面に追加すればネイティブアプリのように1タップで起動できます。

## 特長

- **記録ボタン**：1タップで次の番号と現在時刻を自動追加
- **待機 / 昼休憩ボタン**：番号を振らずに時刻だけ記録（採番は自動でスキップ）
- **押し忘れの修正**：行の「＋」で間に行を挿入し、時刻を手入力。番号は自動で再採番されます
- **編集・削除**：各行の時刻・種別の修正、行の削除に対応
- **日付切替**：前日／翌日ボタンで過去の記録を閲覧・編集
- **コピー / 共有**：元のメモと同じ「番号␣時刻」形式で出力。共有はAndroid標準の共有シート（Bluetooth・ニアバイシェア・LINE等）に対応
- **端末内保存**：データはブラウザのlocalStorageに保存。通信は一切行いません

## 使い方（Android）

1. 公開URLをChromeで開く
2. Chromeメニュー →「ホーム画面に追加」
3. 追加されたアイコンから起動し、「記録」ボタンをタップ

## ファイル構成

- `index.html` — アプリ本体（これ1つで完結）

## 押し忘れたときの直し方

例：番号40を14:45で記録したあと、押し忘れて次のタップが15:12になった場合。

1. 40の行の「＋」を押して下に行を挿入（時刻を手入力）
2. 必要な数だけ挿入する
3. 番号は自動で振り直され、あとから入れた行が41・42、もとの行が43…と整います

## 技術メモ

- 依存ライブラリなし（Vanilla JS / CSS）
- データ保存：`localStorage`
- 共有：Web Share API（`navigator.share`）、非対応環境ではテキスト表示＋コピーにフォールバック
